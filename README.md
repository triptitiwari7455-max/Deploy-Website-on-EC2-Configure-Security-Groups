Deploy Website on EC2 and Configure Security Groups

Overview

This project demonstrates how to deploy a website on an AWS EC2 instance and configure Security Groups to allow web traffic.

Technologies Used

AWS EC2
Apache Web Server
Linux
Security Groups
Steps Performed

Launched an EC2 instance.
Created and used a Key Pair.
Configured Security Group rules.
Installed Apache Web Server.
Deployed website files.
Verified website accessibility through the public IP address.
Security Group Configuration

SSH (Port 22)
HTTP (Port 80)
Learning Outcomes

EC2 Instance Management
Linux Server Administration
Apache Web Server Setup
AWS Security Group Configuration
Website Deployment on Cloud
